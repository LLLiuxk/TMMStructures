### 2. 本次主要更新 (Major Updates)

#### **A. 全量网格采样引擎 (`generate_dataset.py`)**
- **全拓扑枚举**：实现了 `enumerate_all_topologies` 函数，遍历 $2^{n(n-1)/2}$ 种边组合，严格筛选所有符合连通性和度数限制的合法拓扑。
- **笛卡尔积组合**：采样逻辑升级为完全确定性的全排列组合，覆盖：
    - 节点数量分配 ($n_1 \sim n_4$)
    - 所有的合法拓扑邻接矩阵
    - 所有的连接类型组合 (`allowed_connection_types`)
    - 所有的节点位置与宽度步长组合
- **确定性保证**：移除了 `grid` 模式下的所有随机因子，确保数据集生成绝对可复现且不受全局种子影响。

#### **B. 自动化集成流水线 (`generate.bat`)**
- **一键化流程**：通过 `generate.bat` 实现了“生成图像 -> 性质计算 -> 绘制雷达图 -> 汇总分析”的闭环。
- **实时均质化**：在生成每张 PNG 的同时，调用 `homogenize.py` 接口计算其有效力学/热学性质矩阵，并实时存入 `dataset_schema.json`。
- **雷达图导出**：每个样本均自动配套生成性质雷达图 (`_radar.png`)，展示 $E(\theta)$ 和 $\kappa(\theta)$ 的各向异性。

#### **C. 性质覆盖面分析 (`plot_property_coverage.py`)**
- **Hashin-Shtrikman Bounds**：实现了 2D 平面应力下的 HS 上界计算，用于评估生成结构对材料物理性能极限的逼近程度。
- **性能分布可视化**：
    - 绘制 $C_{11} - \phi$ (刚度 vs 体积分数) 散点图，并叠加 HS 上界曲线。
    - 绘制 $C_{11} - C_{22}$ 散点图，直观展示微结构的各向异性覆盖范围。
- **健壮性优化**：增加了 JSON 记录缺失字段的自动过滤和报错拦截机制。
