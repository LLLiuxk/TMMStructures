# TMMStructures - 力热耦合二维微结构参数化逆向生成

## 项目总体目标（Pipeline）

1. **微结构参数化生成** — 根据生成规则和一系列参数确定微结构的拓扑（接触点、连接关系、线条类型和宽度等）
2. **力热均质化计算** — 对生成的微结构快速计算有效力学和热学性质矩阵 ✅ **已完成**
3. **数据集与扩散模型训练** — 构建大规模数据集，训练扩散模型实现逆向生成 ⬜ 待开始
4. **逆向生成** — 给定目标性质矩阵，生成满足要求的微结构 ⬜ 待开始

## 已完成模块：力热均质化 (`homogenize.py`)

### 功能

从 `Input/figures/` 读取二值 PNG 图像（黑色=实体，白色=空洞，仅保存左上 1/4），通过像素级有限元+周期性边界条件计算：

- **有效弹性刚度矩阵 C_eff** (3×3, Voigt 记法: [σxx, σyy, σxy] = C_eff [εxx, εyy, γxy])
- **有效热导率张量 κ_eff** (2×2)

### 技术方案

- 每个像素 → 一个双线性四边形 (Q4) 单元
- SIMP 材料插值（penal=3, Emin=1e-9）
- 周期性边界条件通过 DOF 缩减实现（右→左、下→上、角点统一）
- 全面向量化 (numpy/scipy)，256×256 网格约 20s/张

### 材料参数（当前使用归一化值）

- E0=1.0, ν=0.3, k0=1.0（实体）
- Emin=1e-9, kmin=1e-9（空洞）
- 平面应力假设

### 验证结果

| 测试 | 结果 |
|------|------|
| 纯实体块 → 恢复材料本构矩阵 | ✅ 误差 < 1e-30 |
| 纯空洞块 → 近零性质 | ✅ |
| 棋盘格 → C11=C22, 对称 | ✅ |
| 3 张样本图像处理 | ✅ 结果物理合理 |

### 使用方式

```bash
python homogenize.py          # 批量处理 Input/figures/ 中的所有 PNG
python homogenize.py --test   # 运行验证测试
```

## 已完成模块：微结构参数化与自动生成 (`generate_microstructure.py` & `generate_dataset.py`)

### 系统架构解耦 (Separation of Concerns)

系统实现了“生成策略”与“底层渲染绘制”的完全解耦：

- **`generate_dataset.py` (策略局)**：专注生产抽象的网格/随机结构配置图（`schema`字典）。
- **`generate_microstructure.py` (制造厂)**：负责根据接收到的确定性字典，精准无偏差地处理像素映射和贝塞尔几何形变。

### 拓扑几何生成方案

- **可变边界节点分配**：四条边界($E_1 \sim E_4$)可以被动态分配不同数量的节点 `nodes_per_edge`。节点拥有**中心位置** $t \in (0,1)$ 与**截断厚度** $d \in (0,1)$。
- **纯粹的邻接矩阵图**：利用全局对称邻接矩阵决定连通性，并通过 BFS 广度优先搜索**严格剔除非单连通（有断层/孤岛）**的图。
- **四种不同的连接几何形态 (Connection Types)**：
  1. `straight_line`: 经典直线四边形桁架。（注：同边点连接时，底将自动退化为安全的 `bezier_curve` 避免零厚度直线）。
  2. `tapered_line`: 内缩控制点的贝塞尔曲线，形成“两头粗、中间细”的骨干。
  3. `bezier_curve`: 运用端点边界法向量强制生成平滑弯曲圆弧切线。已修正同边界的两点连接产生顶部过细的问题，赋予独立的推力厚度。
  4. `circular_arc`: 采用二次 Bezier 逼近的完整半圆形/圆轨道。

### 批量生成与双轨驱动配置 (`dataset_config.json`)

系统可以通过自动化参数配置文件 `dataset_config.json` 触发两种完全不同的采样引擎：

1. **随机采样模式 (`sampling_mode: "random"`)**：在范围（range）内使用均匀分布产生完全不确定的随机采样组合。
2. **多维嵌套穷举网格扫描 (`sampling_mode: "grid"`)**：通过给定的 `_step` 步长，采用 `itertools.product` 函数进行严格的笛卡尔积乘法。此模式能够**完全穷举遍历所有边界节点数的高维非对称分配**，并且覆盖所有坐标、宽度的有规律性递进排布，极其适合逆设计网络训练中学习微结构的同构与旋转等变特征。

```json
{
    "sampling_mode": "grid",
    "num_samples": 20,
    "output_dir": "Output/dataset",
    "node_position_range": [0.1, 0.9],
    "node_position_step": 0.2,
    "node_width_range": [0.15, 0.30],
    "node_width_step": 0.1,
    "nodes_per_edge_range": [1, 1],
    "allowed_connection_types": ["bezier_curve"]
}
```
