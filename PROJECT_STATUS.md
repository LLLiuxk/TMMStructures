# TMMStructures - 力热耦合二维微结构参数化逆向生成

## 项目总体目标（Pipeline）

1. **微结构参数化生成** — 根据生成规则和一系列参数确定微结构的拓扑（接触点、连接关系、线条类型和宽度等）
2. **力热均质化计算** — 对生成的微结构快速计算有效力学和热学性质矩阵 ✅ **已完成**
3. **数据集与扩散模型训练** — 构建大规模数据集，训练扩散模型实现逆向生成 ⬜ 待开始
4. **逆向生成** — 给定目标性质矩阵，生成满足要求的微结构 ⬜ 待开始

## 已完成模块：力热均质化 (`homogenize.py`)

### 功能

从 `Input/figures/` 读取二值 PNG 图像（黑色=实体，白色=空洞，仅保存左上 1/4），通过像素级有限元+周期性边界条件计算：

- **有效弹性刚度矩阵 C_eff** (3×3, Voigt 记法: [σxx, σyy, σxy] = C_eff [εxx, εyy, γxy])
- **有效热导率张量 κ_eff** (2×2)

### 技术方案

- 每个像素 → 一个双线性四边形 (Q4) 单元
- SIMP 材料插值（penal=3, Emin=1e-9）
- 周期性边界条件通过 DOF 缩减实现（右→左、下→上、角点统一）
- 全面向量化 (numpy/scipy)，256×256 网格约 20s/张

### 材料参数（当前使用归一化值）

- E0=1.0, ν=0.3, k0=1.0（实体）
- Emin=1e-9, kmin=1e-9（空洞）
- 平面应力假设

### 验证结果

| 测试 | 结果 |
|------|------|
| 纯实体块 → 恢复材料本构矩阵 | ✅ 误差 < 1e-30 |
| 纯空洞块 → 近零性质 | ✅ |
| 棋盘格 → C11=C22, 对称 | ✅ |
| 3 张样本图像处理 | ✅ 结果物理合理 |

### 使用方式

```bash
python homogenize.py          # 批量处理 Input/figures/ 中的所有 PNG
python homogenize.py --test   # 运行验证测试
```

## 已完成模块：微结构参数化与自动生成 (`generate_microstructure.py` & `generate_dataset.py`)

### 技术方案

本模块针对拓扑微结构的特征维度进行了极大的降维，定义了一套能够被机器学习算法（如扩散模型）快速采纳的**参数化规则**：

- **边界节点宽度化**：抛弃了通过连线赋予宽度的传统做法。规定 1/4 微结构的四条边界($E_1 \sim E_4$)上**各且仅有 1 个节点**，并且该节点绑定**中心位置** $t \in (0,1)$ 与**截断厚度** $d \in (0,1)$。
- **纯粹的邻接矩阵图**：利用一个 $4 \times 4$ 对称邻接矩阵，决定 4 个边界点的连接关系，并在生成时进行 BFS 广度优先搜索，**严格剔除任何非单连通（有断层/孤岛）**的图，以及任何节点连接度超过 2 的不合理图。
- **四种不同的连接几何形态 (Connection Types)**：
  1. `straight_line`: 经典直线四边形桁架。
  2. `tapered_line`: 使用两端挤压控制点的贝塞尔曲线，形成“两头粗、中间细”的骨干（可用于轻量化仿生结构）。
  3. `bezier_curve`: 运用端点边界法向量，强制生成垂直且平滑弯曲的圆弧切线（保障周期性平滑过渡）。
  4. `circular_arc`: 采用 $\kappa=0.55228$ 近似的标准 90 度圆形切角轨道。

### 测试结果

- ✅ 脚本 `generate_microstructure.py` 可以正确渲染包含以上四种类型的 `128x128` 实体图片，十字口无生硬断层。
- ✅ 自动化脚本 `generate_dataset.py` 能够随机混抽这些连接类型，顺利且不间断地生成指定数量（如 `num_samples=5`）的符合几何硬性限制的数据集并建立 JSON 元数据记录。
