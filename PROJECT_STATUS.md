# TMMStructures - 力热耦合二维微结构参数化逆向生成

## 项目总体目标（Pipeline）

1. **微结构参数化生成** — 根据生成规则和一系列参数确定微结构的拓扑（接触点、连接关系、线条类型和宽度等）
2. **力热均质化计算** — 对生成的微结构快速计算有效力学和热学性质矩阵 ✅ **已完成**
3. **数据集与扩散模型训练** — 构建大规模数据集，训练扩散模型实现逆向生成 ⬜ 待开始
4. **逆向生成** — 给定目标性质矩阵，生成满足要求的微结构 ⬜ 待开始

## 已完成模块：力热均质化 (`homogenize.py`)

### 功能

从 `Input/figures/` 读取二值 PNG 图像（黑色=实体，白色=空洞，仅保存左上 1/4），通过像素级有限元+周期性边界条件计算：

- **有效弹性刚度矩阵 C_eff** (3×3, Voigt 记法: [σxx, σyy, σxy] = C_eff [εxx, εyy, γxy])
- **有效热导率张量 κ_eff** (2×2)

### 技术方案

- 每个像素 → 一个双线性四边形 (Q4) 单元
- SIMP 材料插值（penal=3, Emin=1e-9）
- 周期性边界条件通过 DOF 缩减实现（右→左、下→上、角点统一）
- 全面向量化 (numpy/scipy)，256×256 网格约 20s/张

### 材料参数（当前使用归一化值）

- E0=1.0, ν=0.3, k0=1.0（实体）
- Emin=1e-9, kmin=1e-9（空洞）
- 平面应力假设

### 验证结果

| 测试 | 结果 |
|------|------|
| 纯实体块 → 恢复材料本构矩阵 | ✅ 误差 < 1e-30 |
| 纯空洞块 → 近零性质 | ✅ |
| 棋盘格 → C11=C22, 对称 | ✅ |
| 3 张样本图像处理 | ✅ 结果物理合理 |

### 使用方式

```bash
python homogenize.py          # 批量处理 Input/figures/ 中的所有 PNG
python homogenize.py --test   # 运行验证测试
```
